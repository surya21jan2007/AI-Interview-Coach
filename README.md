# 🎤 AI Interview Coach

An AI-powered Interview Preparation Assistant built using Python and Streamlit.

The application helps users practice interviews, evaluate responses, calculate interview readiness scores, and generate learning roadmaps based on their target role.

---

## 🚀 Features

* Job Role Selection
* Difficulty Selection
* Interview Question Generation
* Answer Evaluation
* Interview Readiness Score
* Improvement Suggestions
* Learning Roadmap
* Interactive Dashboard

---

* Streamlit
